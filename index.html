The Science of Remembering
Evidence-based techniques for long-term retention — not cramming
🔁

Spaced Repetition

Review material at increasing intervals: 1 day → 3 days → 7 days → 14 days → 30 days. Each review before you forget strengthens the memory trace exponentially. This app tracks your cards to recommend reviews at the right time.

Highest impact
🧩

Active Recall

Don't re-read — test yourself. Retrieving information from memory strengthens it far more than passive review. Every flashcard flip, every quiz question, every "brain dump" is active recall. Re-reading gives you fluency illusions without retention.

Core technique
🔗

Elaborative Interrogation

For every concept, ask: "Why is this true? How does this connect to what I already know?" Connecting A&P 1 to A&P 2 concepts creates richer memory networks. Example: Why does smooth ER detoxify? (lipid-soluble enzymes → membrane affinity)

Deep learning
🎨

Dual Coding

Pair verbal information with visual diagrams. When you read "sarcomere" AND see the A/I band diagram, you create two memory traces that reinforce each other. The diagrams on these flashcards are there for this exact reason — don't skip them.

Visual memory
😴

Sleep & Consolidation

Memory consolidation happens during sleep — especially slow-wave and REM sleep. Study in the evening, sleep 7–9 hours, and review the next morning. Pulling an all-nighter destroys consolidation. Short naps (20–30 min) after studying measurably improve retention.

Non-negotiable
🧱

Interleaving

Don't study all of one chapter, then all of another. Mix topics in each session. It feels harder — that's the point. "Desirable difficulty" forces your brain to discriminate between concepts, building stronger retrieval cues. Use the Mixed mode in quiz.

Counterintuitive
🗣️

The Feynman Technique

Pick a concept. Explain it out loud as if teaching a 10-year-old, using no jargon. Wherever you stumble, that's your knowledge gap. Go back, learn it, explain again. This technique exposes shallow understanding faster than any test.

Self-teaching
📍

Method of Loci

Assign concepts to specific locations in a familiar place (your house, your walk to class). "Walk through" it mentally to retrieve. Great for ordered lists — cranial nerve order (I–XII), layers of the epidermis, vertebral levels. Spatial memory is ancient and powerful.

Memory palace
✍️

Write It By Hand

Handwriting activates more of the brain than typing. For complex diagrams (neuron, sarcomere, nephron), draw and label them from memory repeatedly. Each time you redraw it, you're doing active recall AND dual coding simultaneously.

Proven technique
