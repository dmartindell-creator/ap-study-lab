# A&P Study Lab 🔬

A complete, interactive Human Anatomy & Physiology 1 review app to prepare for A&P 2. Built as a single HTML file — no install needed.

## Features

- **📋 Study Notes** — All major A&P 1 topics with inline SVG diagrams and memory mnemonics
- **🃏 Flashcards** — Flip cards with visual diagrams, definitions, and memory tricks. Filter by chapter. Mark as Mastered or Needs Review.
- **🧠 Quiz Mode** — Multiple choice quizzes: mixed, by chapter, weak spots only, or speed round
- **📊 Dashboard** — Track progress across chapters with mastery percentages
- **🔬 Study Science** — Evidence-based memory techniques: spaced repetition, active recall, the forgetting curve, Feynman technique, and more
- **💾 Progress saved** — Uses localStorage to save your mastered cards and scores between sessions

## Topics Covered (A&P 1 Review)

1. Organization & Basic Chemistry
2. Cell Biology (organelles, transport, division)
3. The Four Tissue Types
4. Integumentary System
5. Skeletal System
6. Muscular System (sarcomere, sliding filament theory)
7. Nervous System (neurons, brain regions, ANS)
8. Homeostasis & Feedback

## How to Use

### Locally
Just open `index.html` in any modern web browser. No server, no install, no dependencies.

### Host on GitHub Pages (free)
1. Create a new GitHub repository
2. Upload `index.html` to the repo
3. Go to **Settings → Pages**
4. Under "Source", select **main branch** and click Save
5. Your app will be live at `https://yourusername.github.io/your-repo-name`

## Study Tips Built Into This App

The app is designed around the most effective memory science:

- **Spaced repetition**: Use the "Got It / Needs Review" buttons and come back to flagged cards
- **Active recall**: Quiz yourself before re-reading notes
- **Dual coding**: Every major term has a diagram to pair visual + verbal memory
- **Interleaving**: Use "Mixed Review" quiz mode to avoid blocking
